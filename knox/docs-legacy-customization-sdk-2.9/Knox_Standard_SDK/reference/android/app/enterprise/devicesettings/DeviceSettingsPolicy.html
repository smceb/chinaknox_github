public boolean allowNFCStateChange (boolean allow)

Since: API Level MDM_5.0

API to allow or disallow the user to change NFC state (turn on or off).

Parameters
allow true to allow NFC change, false to disallow
Returns
  • true if successful, else false
Throws
SecurityException If caller does not have required permissions
Usage
An administrator can use this API to allow or disallow the user to change the NFC state. If set to false (disable), the user will not be able to change NFC state. The S-Beam feature which depend on this policy will not be affected by this setting. This feature does not change the current state of the NFC Adapter. To change NFC state, check startNFC(boolean). 

 mDeviceSettings = DeviceSettingsPolicy.getInstance(mContext);
 try {
     boolean result = mDeviceSettings.allowNFCStateChange(true);
     if (true == result) {
         // NFC state change is not allowed by user.
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }
Permission
The use of this API requires the caller to have the "android.permission.sec.MDM_HW_CONTROL" permission with a protection level of signature.

public int setGlobalProxy (ProxyProperties properties)

Since: API Level MDM_5.5

API to set a Global Proxy on the device.

Parameters
properties ProxyProperties object with proxy information. Use null to remove global proxy.
Returns
  • 1 if policy was successfully set; 0 otherwise.
Throws
SecurityException If caller does not have required permissions
Usage
Administrator will be able to set a Global Proxy on the device. Such proxy will work both for Wifi and Data connections. This proxy may not be considered by some applications (e.g. applications that directly open a socket with the final server), although it is used by most of them. Applications might retry connections without using the global proxy whenever the proxy is unreachable or the authentication fails. 
 DeviceSettingsPolicy mDeviceSettingsPolicy = DeviceSettingsPolicy.getInstance(mContext);
 try {
     // 1. static proxy configuration
     String hostName = "128.128.128.128";
     int portNumber = 80;
 
     List<String> exclusionList = new ArrayList<String>();
     exclusionList.add("102.103.22.202");
     exclusionList.add("www.samsung.com");
 
     ProxyProperties staticProxy = new ProxyProperties();
     staticProxy.setHostname(hostName);
     staticProxy.setPortNumber(portNumber);
     staticProxy.setExclusionList(exclusionList);
 
     // optional, set up authentication for the given static proxy
     staticProxy.setAuthConfigList(Arrays.asList(new AuthConfig[] {
           new AuthConfig("user", "pass")
     }));
 
     int resultStatic = mDeviceSettingsPolicy.setGlobalProxy(staticProxy);
 
     // 2. PAC proxy configuration
     String pacFileUrl = "http://128.128.128.128/proxy.pac";
 
     ProxyProperties automaticProxy = new ProxyProperties();
     automaticProxy.setPacFileUrl(pacFileUrl);
 
     // optional, set up authentication for the multiple proxies that could be returned in PAC file
     automaticProxy.setAuthConfigList(Arrays.asList(new AuthConfig[] {
           new AuthConfig("128.128.128.128", "3128", "user", "pass")
           new AuthConfig("128.128.128.128", "3129", "user1", "pass1")
           new AuthConfig("128.128.128.123", "3128", "user2", "pass2")
     }));
 
     int resultAutomatic = mDeviceSettingsPolicy.setGlobalProxy(automaticProxy);
 
 } catch (SecurityException e) {
      Log.w(TAG, "SecurityException: " + e);
 }
Permission
The use of this API requires the caller to have the "android.permission.sec.MDM_SECURITY" permission which has a protection level of signature.

public boolean startNFC (boolean start)

Since: API Level MDM_5.0

API to turn NFC on or off.

Parameters
start true to start NFC, false to stop NFC
Returns
  • true if successful, else false
Throws
SecurityException If caller does not have required permissions
Usage
An administrator can use this API to turn NFC on or off. If set to false, NFC will be turned off. If set to true, NFC will be turned on. The S-Beam feature which depends on this policy will also be affected by this setting. 

 
 mDeviceSettings = DeviceSettingsPolicy.getInstance(mContext);
 try {
     boolean result = mDeviceSettings.startNFC(true);
     if (true == result) {
         // NFC adapter is turned on
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }
Permission
The use of this API requires the caller to have the "android.permission.sec.MDM_HW_CONTROL" permission with a protection level of signature.
