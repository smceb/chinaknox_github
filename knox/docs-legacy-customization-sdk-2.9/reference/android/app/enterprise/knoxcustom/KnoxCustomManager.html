Class Overview

This class provides APIs to enable, disable and control behavior of the device for Knox Custom Manager.

NOTE: Since version 2.5, the architecture of the SDK changed.
If your device supports version 2.5 or above, please use the CustomDeviceManager class.

public int copyAdbLog (int logType)

Since: API Level Knox Customization_2.4

This method copies log of specified type to /data/log.

Parameters
logType LOGCAT_LOG or KERNEL_LOG.
For type LOGCAT_LOG log file will be copied to /data/log/customLogcat.log.
For type KERNEL_LOG log file will be copied to /data/log/customKernel.log.
Returns
  • SUCCESS The log was successfully copied.
    NOT_SUPPORTED The specified log type is not available.
Throws
SecurityException The calling application does not have the required permission.
Usage
 try {
     KnoxCustomManager kcm = KnoxCustomManager.getInstance();
     kcm.copyAdbLog(KnoxCustomManager.LOGCAT_LOG);
 } catch(SecurityException e) {
     Log.w(TAG, "SecurityException:" + e);
 }
Permission
The use of this API requires the caller to have the "com.sec.enterprise.knox.permission.CUSTOM_SYSTEM" permission.
Since
Version 2.4
Multiuser Environment
Global Scope
