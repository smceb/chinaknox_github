Class Overview

This class contains a set of string constants which serve as keys to provide the parameters needed for a SAML request to the GenericSSO framework. These keys can be used in a map and provided as arguments to enrollSSOVendor and addWhiteListPackages methods in the GenericSSO class. MDM administrators have to provide these values based on the requirements of the SSO provider that has been configured.
