public int allowBTSecureAccess (boolean allowAccess)

Since: API Level KNOX_2.0

API to enable usage of bluetooth secure access for smartcard usage. The administrator needs to set the Bluetooth secure mode to true using the android.app.enterprise.BluetoothSecureModePolicy.enableSecureMode(BluetoothSecureModeConfig configObj, List whiteList) policy in order to allow bluetooth secure access for smartcard usage. If Bluetooth Secure mode is not already enabled at device level this API will return false.
Note: Since KNOX 2.4, Bluetooth Secure Mode is not required to be enabled at device level. 

 The bluetooth device white list configuration 

 Name: bai2ndGen
 Cod: 0x00001f00 (7936) (uncategorized)
 UUIDs: 00001101-0000-1000-8000-00805f9b34fb (SPP) and 00001200-0000-1000-8000-00805f9b34fb (PnP)

Parameters
allowAccess boolean which will allow or disallow usage of Bluetooth Secure Access for smartcard usage
Returns
  • allowBTSecureAccess error codes:
    ALLOW_BT_SECURE_ACCESS_SUCCESS if allowBTSecureAccess is successful
    ALLOW_BT_SECURE_ACCESS_FAILURE if allowBTSecureAccess is failure
    BT_SECURE_MODE_DISABLED if bluetooth secure mode is disabled. Since KNOX 2.4 this error code is not supported.
    ALLOW_BT_SECURE_ACCESS_NOT_APPLICABLE if allowBTSecureAccess is not applicable for this user
Throws
SecurityException If caller does not have required permissions
Usage
This API should be used to enable or disable usage of bluetooth secure access for smartcard usage. 

 EnterpriseKnoxManager ekm = EnterpriseKnoxManager.getInstance();
 try {
     KnoxContainerManager containerMgr = ekm.getKnoxContainerManager(Context context, int containerId);
     SmartCardAccessPolicy smartcardAccessPolicy = containerMgr.getSmartCardAccessPolicy();     
     int ret = smartcardAccessPolicy.allowBTSecureAccess(true);
 } catch (SecurityException e) {
     Log.e(TAG, "SecurityException: " + e);
 }

Permission
The use of this API requires the caller to have the "android.permission.sec.MDM_SMARTCARD" permission which has a protection level of system.
Since
KNOX 2.0
