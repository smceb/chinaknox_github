public boolean disableAuthentication ()

Since: API Level KNOX_1.0

This method is deprecated.
Not supported

API to disable SmartCard authentication for Browser.

Returns
  • true if success, else false
Throws
SecurityException If caller does not have required permissions
Usage

An administrator can disable SmartCard authentication for Browser.

 
 SmartCardBrowserPolicy policy = SmartCardBrowserPolicy.getInstance();
 
 try {
 
     boolean result = policy.disableAuthentication();
     if (result) {
         // disabled
     } else {
         // Failed
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }
For Container:
 // When you create container successfully, containerID will be returned via intent.
 // Use this containerID in below API.
 EnterpriseKnoxManager ekm = EnterpriseKnoxManager.getInstance();
 KnoxContainerManager kmcm = ekm.getKnoxContainerManager(mContext, containerID);
 SmartCardBrowserPolicy policy = kmcm.getSmartCardBrowserPolicy();
 try {
 
     boolean result = policy.disableAuthentication();
     if (result) {
         // enabled
     } else {
         // Failed
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }

Permission
The use of this API requires the caller to have the "android.permission.sec.MDM_SMARTCARD" permission which has a protection level of signature.

public boolean isAuthenticationEnabled ()

Since: API Level KNOX_1.0

This method is deprecated.
Not supported

API to check whether smartcard authentication is enabled for Browser.

Returns
  • true if smartcard authentication is enabled for Browser else false
Usage
An administrator can use this API to check whether smartcard authentication is enabled for Browser. The default value is false (disabled).

For a device managed by multiple administrators, this method returns true (enabled) if at least one administrator has enabled smartcard authentication for Browser.



NOTE: Since KNOX 2.6, this policy is no longer supported by Chrome.

API to enforce a particular SmartCard certificate for a particular host URL

Returns
  • true if success, else false

An administrator can enforce or invoke SmartCard use of a particular SmartCard credentials for given Host URL.
