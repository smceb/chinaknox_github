public abstract void updateStatus (int errorcode, Bundle data)

Since: API Level KNOX_1.0

This method is deprecated.
Since KNOX 2.0

API to handle callback mechanism in Enterprise Container Policy

Parameters
errorcode It can contain one of the constants declared in this class
data Bundled data.
Usage
This callback is being used in APIs such as:
  • createContainer(EnterpriseContainerCallback)
  • removeContainer(int, EnterpriseContainerCallback)
  • installPackage(String, int, EnterpriseContainerCallback)
  • uninstallPackage(String, EnterpriseContainerCallback)

 EnterpriseContainerCallback mEnterpriseContainerCallback = new EnterpriseContainerCallback(){
      Override
      public void updateStatus(int errorcode, Bundle data){
          Log.v(TAG, "package installation ended with errorCode: " + errorcode);
      }
 }

 // When you create container successfully, containerID will be returned via intent.
 // Use this containerID in below API.
 EnterpriseKnoxManager ekm = EnterpriseKnoxManager.getInstance();
 try {
     EnterpriseContainerManager containerMgr = ekm.getEnterpriseContainerManager(containerID);
     ContainerApplicationPolicy cap = containerMgr.getContainerApplicationPolicy();
     int installType = ContainerApplicationPolicy.INSTALL_ONLY;
     boolean result = cap.installPackage(packagePath, installType, mEnterpriseContainerCallback);
 } catch (IllegalArgumentException e) {
     Log.w(TAG, "IllegalArgumentException: " + e);
 }
