public boolean requireCredentials (String emailAddress, boolean require)

Since: API Level KNOX_1.0

This method is deprecated.
Not supported

API to enforce or invoke use of SmartCard credentials for an Email account.

Parameters
emailAddress Email address of the account
require true if SmartCard credentials required, false if does not require
Returns
  • true if success, else false
Throws
SecurityException If caller does not have required permissions
Usage

An administrator can enforce or invoke SmartCard use of SmartCard credentials for given Email account. 

 SmartCardEmailPolicy policy = SmartCardEmailPolicy.getInstance();
 
 try {
     String emailAddress = "aa@xyz.com";
     boolean result = policy.requireCredentials(emailAddress, true);
     if (result) {
         // Success: enforced use of SmartCard credentials on this email account
     } else {
         // Failed: enforcing use of SmartCard credentials on this email account
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }
For Container:
 // When you create container successfully, containerID will be returned via intent.
 // Use this containerID in below API.
 EnterpriseKnoxManager ekm = EnterpriseKnoxManager.getInstance();
 KnoxContainerManager kmcm = ekm.getKnoxContainerManager(mContext, containerID);
 SmartCardEmailPolicy policy = kmcm.getSmartCardEmailPolicy();
 try {
     String emailAddress = "aa@xyz.com";
     boolean result = policy.requireCredentials(emailAddress, true);
     if (result) {
         // Success: enforced use of SmartCard credentials on this email account
     } else {
         // Failed: enforcing use of SmartCard credentials on this email account
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 }

Permission
