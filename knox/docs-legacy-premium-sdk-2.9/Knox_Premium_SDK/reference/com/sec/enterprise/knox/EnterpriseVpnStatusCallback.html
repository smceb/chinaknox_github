public abstract void updateStatus (String profileName, int statusCode, String statusString)

Since: API Level KNOX_1.0

This method is deprecated.
Since KNOX 2.7

API to handle callback mechanism in Enterprise VPN Policy.

Parameters
profileName The profileName which is being removed.
statusCode The values returned are

3 which implies that the profileName is in Disconnecting state.
6 which implies that the profileName has been removed successfully.

The administrator will be getting the status code 3 followed by status code 6.
statusString The status of the Removed Profile and the values returned are

"Cancelled" which correponds to the Disconnecting state.
"No Error" which corresponds to the Removed state.
Usage
 EnterpriseVpnStatusCallback vpnStatusCallback = new EnterpriseVpnStatusCallback(){
      Override
      public void updateStatus(String profileName, int statusCode, String statusString) {
          Log.v(TAG, "VPN remove connection ended with
                statusCode: " + statuscode + " statusString = " + statusString);
      }
 }
