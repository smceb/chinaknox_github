Class Overview

This class provides references to the constants and return values used in the GenericSSO framework along with a set of keys to all client applications and authenticators to obtain access to configuration data set by the MDM administrator. The keys in this class present the comprehensive set of values obtained as response when an authorization is requested through the OAuth and SAML protocols based single sign-on. The applications can make use of the entire set or a subset of the keys present here and populate a bundle which will then be used by the authenticators to extract information about the client application as presented by the MDM administrator.
