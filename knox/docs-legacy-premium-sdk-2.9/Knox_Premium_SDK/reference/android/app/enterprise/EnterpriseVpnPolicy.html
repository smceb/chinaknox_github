public boolean setEnterpriseVpnConnection (EnterpriseVpnConnection vpn, String oldName)

Since: API Level

This method is deprecated.
Since MDM 5.7

API to set the object that will manipulate an enterprise VPN connection.
This is the first step for creating VPN Connection.

Parameters
vpn Enterprise VPN connection profile.
oldName Name of the connection to be changed or null for a new network.
Returns
  • true if the connection is added or edited successfully, false if not.
Throws
SecurityException If caller does not have required permissions
UnsupportedOperationException The API is not supported by the VPN Type Connection
Usage

 
 EnterpriseDeviceManager edm = (EnterpriseDeviceManager) getSystemService(EnterpriseDeviceManager.ENTERPRISE_POLICY_SERVICE);
 // Create the profile
 EnterpriseVpnConnection entVpn = new EnterpriseVpnConnection();
 entVpn.name = "CompanyVpn";
 entVpn.host = "vpn.company.com";
 entVpn.type = EnterpriseVpnPolicy.VPN_TYPE_ANYCONNECT;
 // For Automatic mode
 entVpn.setCertAuthMode(EnterpriseVpnPolicy.VPN_CERT_TYPE_AUTOMATIC);
 // For Manual mode
 entVpn.setCertAuthMode(EnterpriseVpnPolicy.VPN_CERT_TYPE_MANUAL);
 entVpn.certCommonName(readCertCn("company_cert")); // assume built-in method
 entVpn.certHash(readCertHash("company_cert"));     // assume built-in method
 // Set the profile
 try {
     boolean success = edm.getEnterpriseVpnPolicy().setEnterpriseVpnConnection(entVpn, null));
     if (success) {
         Log.d(TAG, "Set EnterpriseVpnConnection is Success");
     } else {
         Log.d(TAG, "Set EnterpriseVpnConnection is Failure");
     }
 } catch (SecurityException e) {
     Log.w(TAG, "SecurityException: " + e);
 } catch (UnsupportedOperationException e) {
     Log.w(TAG, "UnsupportedOperationException: " + e);
 }
